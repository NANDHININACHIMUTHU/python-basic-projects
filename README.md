
# Python Basic Programs
This repository contains beginner Python programs built to practice
core Python concepts.

## Programs Included
1. Hello World
2. Read name and age
3. Swap two numbers
4. Add two numbers
5. Calculator
6. Celsius to Fahrenheit
7. Kilometer to Miles
8. Area of Circle
9. Area of Rectangle
10. Average of Numbers

## Concepts Used
- Input / Output
- Conditions
- Loops
- Arithmetic Operators
